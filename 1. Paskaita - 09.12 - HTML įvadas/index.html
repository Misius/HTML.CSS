Tag'ų sąrašas

  • h1-h6 - antraštė
  • p - paragrafas
  • img - paveiksliukas
  • a - nuoroda
  • div - Div'as (konteineris)
  • ul - nerikiuotas sąrašas
  • ol - rikiuotas sąrašas
  • li - Sąrašo elementas
  • dl - aprašomasis sąrašas
  • dt - aprašomojo sąrašo terminas
  • dd - aprašomojo sąrašo paaiškinimas
  • hr - horizontali linijas
  • br - eilutės lūžis
  • table - lentelė
  • caption - lentelės pavadinimas
  • colgroup - lentelės stulpelių grupė
  • thead - lentelės galva
  • tbody - lentelės kūnas
  • tfoot - lentelės apačia
  • tr - lentelės eilutė
  • th - lentelės duomenų grupės pavadinimas
  • td - lentelės duomuo
  • form - forma
  • label - etiketė
  • input - įvesties laukas
  • texterea - didelis teksto įvesties laukas
  • button - mygtukas
  • center - centruoja tekstą
  • b - pastorina šriftą
  • strong - pastorina šriftą ir yra geriau skaitomas akliesiems skirtom programom
  • i - padaro tekstą pasvirusį
  • em - padaro tekstą pasvirusį ir kaip strong
  • sup - pakelia tekstą aukštyn
  • sub - nuleidžia tekstą žemyn
  • ins - pabraukia tekstą
  • mark - pažymi tekstą (geltonai)
  • s - perbraukia tekstą
  • small - sumažina tekstą
  • big - padidina tekstą
  • span - tuščias linijinis tag'as

Atributų sąrašas

  • src - šaltinis (nuoroda į failus)
  • href - nuoroda
  • width - plotis
  • height - aukštis
  • onsubmit - formos pateikimas (ką daryti pateikus)
  • for - kam skirta etiketė (nurodo į formos elemento id)
  • name - formos elemento vardas
  • type - įvesties lauko tipas (jų daug)
  • placeholder - įvesties lauko pavyzdinis tekstas
  • onclick - mygtuko pateikimas (ką daryti pateikus)

Klaviatūros trumpinių sąrašas

  • SHIFT ALT F - Viso failo tabuliavimas
  • ctrl x - teksto iškirpimas (kopijavimas ir ištrynimas)
  • ctrl c - teksto kopijavimas
  • ctrl v - teksto įklijavimas
  • ctrl shift p - command palette
  • window v - atidaro clipboard su buvusiais nukopijuotasiai tekstais
  • end - eilutės galas
  • home - eilutės pradžia
  • alt + shift - pakeičia kalbą
  • ctrl f - atidaro paieškos lauką
  • ctrl s - išsaugoja
  • ctrl r arba f5 perkrauną puslapį
  • alt tab - vaikščioti tarp atidarytų programų
  • ctrl a - viską pažymi
  • shift alt - kopijuoja eilutę žemyn
  • ctrl n - naujas langas browser/ naujas langas vs code
  • ctrl t - naujas tabas
  • ctrl y- atšaukia ctrl z veiksmą
  • ctrl <-- - vaikščioja per žodžius
  • windows <-- - nukelia dabartinį langą į dabartinę pusę
