CSS Rašymo būdai

  • Inline - Linijinis rašomas tag'o viduje naudojant style atributą
  • internal - Vidinis rašomas style tag'o viduje (head dalyje)
  • external - Išorinis rašomas atskirame CSS faile, kurį reikia prijungti prie HTML naudojant link tagą

CSS Selektoriai

  • element - Parenkamas elementas pagal jo pavadinimą (p{})
  • id - sukuriamas susigalvotas id pavadinimas ir priskiriamas vienam elementui (#idPavadinimas{})
  • class - sukuriamas susigalvotas klasės pavadinimas ir priskiriamas bet kokiam kiekiui elementų (.klasesVardas{})

CSS Stiliai

  • color - skirtas pakeisti teksto spalvą
  • font-size - skirtas pakeisti teksto dydį
  • background-color - skirtas pakeisti fono spalvą
  • font-family - skirtas pakeisti teksto šriftą
  • text-align - skirtas lygiuoti tekstą
  • font-weight - skirtas pakeisti teksto storumą
  • line-height - skirtas pakeisti eilutės aukštį
  • border - skirtas nurodyti rėmelio storį+tipą+spalvą
  • border-radius - skirtas apvalinti rėmelių kampus
  • font-style - skirtas suteikti tekstui stilių
  • text-decoration = skirtas tekstui dekoruot
  • list-style-type - skirtas pakeisti listo tipą

Spalvos

  • Spalvos vardo naudojimas. Yra 140 spalvų.
  • HSL naudojimas. Niekas jo nenaudoja.
  • HEX naudojimas. Naudojamas taip pat plačiai kaip ir RGB. (#ff0000
  • RGB naudojimas. Naudojamas taip pat plačiai kaip ir HEX. (rgb(255, 0, 0))

Permatomos spalvos

  • HSL permatomumas nurodomas nuo 0 iki 1
  • HEX permatomumas nurodomas nuo 00 (pilnai matomas). (#ff0000ff)
  • RGB permatomumas nurodomas nuo 0 iki 1 (rgba(255, 0, 0, 1))

HEX ir RGB

HEX - kiekvienos spalvos kiekkio nurodai nuo 00 (nesuteikiama spalvos) iki ff (suteikiama spalvos maximaliai)
