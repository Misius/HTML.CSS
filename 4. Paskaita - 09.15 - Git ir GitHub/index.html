  • Static - statinė
  • Relative - reliativoji
  • Absolute - absoliučioji

Naudojant bet kurią apart Statinės pozicijos, gauname prieigą prie 5 naujų stilių: top, lef, right, bottom, z-index.
Top, lef, right, bottom, z-index stiliai nurodo kokį atstumą elementas turi būti atitrauktas.

Statinė pozicija - nieko nedaro negali keisti kas viršuj paminėta.
Z--index stilius nurodo elemento prioritetą sluoksniuose.

Git komandos

  • init - initialize - inicializuoti
  • status - statusas - pateikia informaciją apie esamoje repozitorijoje
  • add - stage'ina/prideda/paruošia - paruošia failus commit'inimui
  • commit - sukuria kodo versiją iš stage'intų failų
  • push - įstumti - lokalius (jau sukomitintus) pakitimus perduoda GitHub'ui
