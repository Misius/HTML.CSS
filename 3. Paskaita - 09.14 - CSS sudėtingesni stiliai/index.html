Puslapio struktūra

  • header - puslapio viršus kuriama nurodoma informacija apie puslapį/meniu
  • nav - skirtas kurti puslapio meniu
  • section - didžiulė puslapio dalis
  • div - maža puslapio dalis
  • main - pagrindinė puslapio dalis
  • article - main viduje esanti sekcija
  • aside - šoninė puslapio sekcija
  • footer - puslapio apačia, kurioje yra įvairi informacija kaip termas and uses, privacy policy, copuright, links, partners...

Menu informacija

Kuriant menu - kuriamas nav tag'as, kurio viduje yra rikiuotas/nerykiutas sąrašas (ol/ul). To sąrašo elementų (li) viduje yra nuorodos tag'as (a), kuris nurodo į kažkurią puslapio dalį arba kitą puslapį.

nuorodos tag'o (a) atidaromojoje dalyje esantis atributas href nurodo į kur vartotojas turi būti nukeltas paspaudus ant nuorodas.

Jeigu atributas nurodo į kažkokį ID (href="#ID_Pavadinimas"), tuomet vartotojas yra nukeliamas į puslapio dalį, kuriai yra suteiktas nurodytas ID (id="ID_Pavadinimas").
Puslapio viršus

Jeigų href atributas nurodomas į kitą failą (href="./aplankalas.html"), tuomet vartotojas yra nukeliamas į nurodytą html failą.

Inline, Block ir Inline-block

Inline

Inline elementai yra tokie elemetai, kurie užima tik tiek vietos ekrane, kiek jiems yra reikalinga. Minusas tas, kad negalima keisti šių elementų dydžių. Pliusas tas, kad lengvai įtelpa tarp kitų informacijos vienetų.
Tag'ų pavuzdžiai: a, span, img, input, button, label, texterea, select, cite...

Block

Block elementai yra tokie elementai, kurie užima visą leistiną plotį ir neįsileidžia kitų elementų į tą pačią eilutę Minusas tas, kad užima visą leistiną plotį. Pliusas tas, kad galima keisti jų dydį.
Tag'ų pavyzdžiai: h1-h6, p, div, section, table, main, article, nav, ol, ul, li, fieldset, blockquote, aside, canvas...

Model box

Margin - Atstumas nuo elemento rėmelio iki aplinkinių elementų rėmelių (skirtingų elementų margin'ai persidengia ties viršum/apačia, ties šonus nepersidengia).

Border - Rėmelis kuris yra tarp Margin ir Padding.

Padding - Atstumas nuo elemento turinio iki aplinkinių elementų rėmelių

Content - Elemento turinys. (blokinio elemento turinys užims visą leidžiamą eilutę arba nurodytą dydį)
